# Lobby

## Deluxhub

Attach an item to teleport 

```yml
survival:
    material: GRASS_BLOCK
    slot: 15
    amount: 1
    glow: false
    display_name: '&aSurvival'
    lore:
      - '&7Join now!'
      - '&aMulti lore support too!'
    actions:
      - '[CLOSE]'
      - '[MESSAGE] &7Sending you to: &eSurvival'
      - '[BUNGEE] survival'
```
